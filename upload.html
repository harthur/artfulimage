Submitting Images for Print

You can drop image files off at the studio anytime we are open.
If you prefer, you can email image files under 5 MB as attachments to brian@theartfulimage.com
If you want to submit a large image or several images and can't drop them off, you can upload them to the Artful Image public FTP server:
ftp://theartfuliamge.com. You will need to email for a username and password to access the server.

Image file formats and recomendations

Acceptable file types: JPEG, TIFF, PSD, PDF.
Almost any color space will be excepted, but ProPhotoRGB, AdobeRGB, and sRGB are preferred.
Please flatten any layers before submitting
Optimum resolution for files is 360 ppi, but we can work with whatever you have. For high quality prints, we recomend
a resolution of at least 240 ppi at the size you want to print.

State whether you want borders or borderless printing
For borderless printing, expect to lose approximately 3 mm (1/8 inch) from each edge.
For bordered prints, please state the size of the borders you require
The default is 1/4 inch borders unless otherwise specified by you.
Borders are internal to the print. An 8 by 12 inch print, for example, will be 7 1/2 by 11 1/2 inches on 8 x 12 inch paper.

Instructions for uploading images from a computer running Windows

On a Windows machine you can upload images by copying and pasting or draging and dropping them into Windows Explorer.
That is Windows Explorer the file manager, not Internet Explorer the browser.

In Windows Explorer:
1. Click on the icon at the left of the path bar to select the current path/address
2. Type in ftp://theartfulimage.com
3. Hit the Enter key on the keyboard
4. You should be prompted for a username and password
5. After entering the username and password, hit the OK or Logon button
6. You should now be able to copy and paste, or drag and drop image files to the server
Note: Depending on your connection speed, it may take several minutes for your files to upload.
